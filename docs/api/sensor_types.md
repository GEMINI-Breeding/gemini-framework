# Sensor Types API

### Description

A sensor type defines the type of a specific [Sensor](sensors.md).

The following sensor types are pre-defined, along with their `sensor_type_id`:


| Sensor Type   | sensor_type_id |
|---------------|----------------|
| Default       | 0              |
| RGB           | 1              |
| NIR           | 2              |
| Thermal       | 3              |
| Multispectral | 4              |
| Weather       | 5              |
| GPS           | 6              |
| Calibration   | 7              |
| Depth         | 8              |
| IMU           | 9              |
| Disparity     | 10             |
| Confidence    | 11             |
