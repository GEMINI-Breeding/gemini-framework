# Dataset Types API

### Description

A dataset type defines the type of a specific [Dataset](datasets.md). 

The following dataset types are pre-defined, along with their `dataset_type_id`:

| Dataset Type | `dataset_type_id` |
|--------------|-------------------|
| Default      | 0                 |
| Sensor       | 1                 |
| Trait        | 2                 |
| Script       | 3                 |
| Model        | 4                 |
| Procedure    | 5                 |
| Other        | 6                 |
