# Experiments API

### Description

An experiment is the core entity of the GEMINI Framework. An experiment can have one or many of the following: 

- [Cultivars](cultivars.md)
- [Datasets](datasets.md)
- [Models](models.md)
- [Plots](plots.md)
- [Procedures](procedures.md)
- [Scripts](scripts.md)
- [Seasons](seasons.md)
- [Sensors](sensors.md)
- [Sites](sites.md)
- [Traits](traits.md)
