# Trait Levels API

### Description

A trait level defines the level at which is a specific [Trait](traits.md) is defined.

The following trait levels are pre-defined, along with their `trait_level_id`:

| Trait Level | ID |
|---|---|
| Default | 0 |
| Plot | 1 |
| Plant | 2 |
